windowsProfiles
===============

Provides a list of :ref:`windowsprofile-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``windowsProfiles`` are:

	* ``windowsProfiles``: the list of :ref:`windowsprofile-object` objects
	* ``total`` (long): the total number of objects
