scanImports
===========

Provides a list of :ref:`scanimport-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``scanImports`` are:

	* ``scanImports``: the list of :ref:`scanimport-object` objects
	* ``total`` (long): the total number of objects
