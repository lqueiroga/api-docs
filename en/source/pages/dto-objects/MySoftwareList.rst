mySoftwareList
==============

Provides a list of :ref:`mysoftware-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``mySoftwareList`` are:

	* ``mySoftwareList``: the list of :ref:`mysoftware-object` objects
	* ``total`` (long): the total number of objects
