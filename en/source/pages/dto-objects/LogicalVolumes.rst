logicalVolumes
==============

Provides a list of :ref:`logicalvolume-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``logicalVolumes`` are:

	* ``logicalVolumes``: the list of :ref:`logicalvolume-object` objects
	* ``total`` (long): the total number of objects
