Blueprint API Resources

	* :ref:`blueprintPipelineStatus-get`: 
	* :ref:`blueprintPipeline-cancel`: Cancel a running preparation

	* :ref:`blueprint-delete`: Removes a blueprint from a user's ``Blueprint Library``
	* :ref:`blueprint-deploy`: Deploys a blueprint

	* :ref:`blueprint-predeploy`: Returns the status of the appliances used for a blueprint
	* :ref:`blueprint-update`: Updates a blueprint from a user's ``Blueprint Library``
	* :ref:`pipeline-get`: 
	* :ref:`pipeline-getAll`: Returns all the preparations created for a blueprint
	* :ref:`prepare`: Prepares a blueprint for deployment
